# TreasureHunt
TreasureHunt is a game based on threejs in which a boy is walked through 2 levels that contain diamonds to be aquired and traps to be avoided. For trial purposes, collecting 5 diamonds counts as a win.
Ideas covered:
* Spline
* textures
* animations
* 3D motions
* threejs text
* user input
* Hemisphere light
* 3D drawing
